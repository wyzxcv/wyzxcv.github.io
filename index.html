电脑软件集合

OBS

官网:https://obsproject.com/
github:https://github.com/obs-studio-windows

介绍:

OBS（Open Broadcaster Software）是一款免费的开源软件，主要用于视频录制和直播。它允许用户将视频和音频源组合成复杂的场景，并能够实时推流到多个平台，如Twitch、YouTube和Facebook Live等。OBS因其强大的功能和灵活性，受到游戏玩家、教育工作者和专业人士的广泛欢迎。

主要功能:

  1. 视频录制：用户可以录制高质量的视频内容，适用于教程、游戏回放和产品评测等。
  2. 直播推流：OBS支持将视频流实时推送到各大直播平台，适合进行在线讲座、活动直播等。] 
  3. 多场景管理：用户可以创建多个场景，方便在直播过程中无缝切换不同的视角和内容。
  4. 音频混音：OBS提供强大的音频控制功能，用户可以管理多个音频源，调整音量和效果。
  5. 插件支持：OBS支持多种插件，用户可以根据需要扩展软件功能，满足不同的直播需求。

Everything

官网:https://www.voidtools.com/zh-cn/downloads/

介绍:

Everything是由澳大利亚开发者David Carpenter创建的一款文件搜索工具。它的主要功能是通过文件和文件夹的名称进行快速搜索，能够实时跟踪文件变化，并且占用极少的系统资源。该软件的体积小巧，界面简洁，适合Windows系统用户使用。

主要功能:

  1. 快速搜索：Everything通过直接访问NTFS磁盘的MTF表，能够在几秒钟内索引数十万甚至上百万个文件。例如，索引20,000个文件只需约1秒钟，索引1,000,000个文件则只需约1分钟。

  2. 低资源占用：该软件在运行时占用的内存非常少，通常在3-5MB之间，适合各种配置的电脑使用。

  3. 实时更新：Everything能够实时监控文件系统的变化，文件和文件夹的名称变化会立即反映到搜索结果中。

  4. 多种搜索功能：用户可以通过输入关键词进行搜索，支持拼音搜索和内容搜索等多种方式，极大提高了搜索的灵活性和准确性。

Bandzip

官网:https://www.bandisoft.com/bandizip/

介绍:

Bandizip 是一款功能强大的 压缩和解压缩软件，支持多种主流压缩格式，如 ZIP、7Z、RAR、ISO 等。它具有快速的压缩和解压缩速度，支持 多线程压缩、分卷压缩和密码保护 等高级功能，界面简洁直观，适合初学者和高级用户使用。Bandizip 是一款 免费软件，同时也提供付费版本以获取更多实用功能。

主要功能:

  1. 压缩:支持的格式: ZIP，ZIPX，7Z(lzma2)，EXE，TAR，TGZ，LZH，ISO，GZ，XZ，ZPAQ，ZIP文件修改（添加/删除/重命名）。
  2. 解压:支持的格式: 7Z, AES, ALZ, ARJ, BH, BIN, BZ, BZ2, CAB, Compound(MSI), EGG, GZ, IMG, ISO, ISZ, LHA, LZ, LZH, LZMA, PMA, RAR, RAR5, SFX(EXE), TAR, TBZ, TBZ2, TGZ, TLZ, TXZ, UDF, WIM, XPI, XZ, Z, ZIP, ZIPX, ZPAQ。
  3. 对ZIP/7z格式可进行分卷压缩。
  4. 加密压缩。

Snipaste

官网:https://zh.snipaste.com/

介绍:

Snipaste 是一款强大的截图和贴图工具，允许用户快速截取屏幕并将截图以浮窗形式显示，极大提升工作效率。

主要功能:

  1. 截图功能:- Snipaste 允许用户快速截取屏幕上的任意区域，支持全屏截图、窗口截图和自定义区域截图。用户只需按下快捷键（如 F1）即可开始截图。
  2. 贴图功能:
    截图后，用户可以将截图以浮窗形式贴在屏幕上，方便对比和查看。贴图可以设置为半透明，用户可以自由调整浮窗的位置和大小。
  3. 编辑和标注:Snipaste 提供丰富的编辑功能，用户可以在截图上添加文字、箭头、标注等，支持调整颜色、画笔宽度和文字大小等。
  4. 剪贴板支持:
    Snipaste 还支持将剪贴板中的文字或颜色信息转化为图片窗口，用户可以对这些内容进行缩放、旋转、翻转等操作。

NDM

官网:https://github.com/Neatdownloadmanager/Neat-Download-Manager/releases

介绍:

NDM 是 Neat Download Manager 的缩写，它是一款轻量级的多线程下载工具，支持 Windows 和 macOS 操作系统。

主要功能:

  1. 多线程下载：能够显著提高下载速度。
  2. 简洁界面：用户界面简单易用，无广告。
  3. 支持多种协议：支持 HTTP、HTTPS、FTP 等协议。
  4. 小巧体积：软件体积小于 1MB，便于快速下载和使用。

Quicklook

官网:https://github.com/QL-Win/QuickLook/releases

介绍:

QuickLook是一款开源、免费的文件快速预览工具，能够在Windows系统上实现类似macOS的空格键预览功能，支持多种文件格式的快速查看。

主要功能:

  1. 多格式支持：QuickLook支持预览多种文件格式，包括图片（如JPG、PNG、GIF、PSD等）、音频（如MP3、AAC等）、视频（如MP4、AVI等）、文档（如PDF、Word、Excel等）和压缩文件（如ZIP）等。
  2. 快速预览：用户只需选中文件并按下空格键，即可快速查看文件内容，无需打开文件。
  3. 低资源占用：QuickLook的内核基于.NET Framework，响应速度快，内存占用低，适合在老旧电脑上运行。
  4. 插件扩展：用户可以通过右键点击QuickLook的任务栏图标获取新插件，进一步扩展其功能。
